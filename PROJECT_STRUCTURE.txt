========================================
üîÑ DATA FLOW
========================================

1. User uploads images
   ‚îî‚îÄ> SatelliteAnalysis.jsx (React)

2. Frontend sends request
   ‚îî‚îÄ> POST /api/satellite/analyze

3. TerraTrack Server (Express)
   ‚îî‚îÄ> Proxy to satellite backend

4. Satellite Backend (FastAPI)
   ‚îî‚îÄ> Receives files

5. Image Converter (if RGB)
   ‚îî‚îÄ> Converts to 13-band format

6. AI Model (Siamese U-Net)
   ‚îî‚îÄ> Detects changes

7. Environmental Analyzer
   ‚îî‚îÄ> Calculates NDVI, NDBI, NDWI

8. LLM Explainer (Gemini)
   ‚îî‚îÄ> Generates insights

9. Visualization Generator
   ‚îî‚îÄ> Creates 12-panel image

10. Results returned
    ‚îî‚îÄ> Displayed in UI
